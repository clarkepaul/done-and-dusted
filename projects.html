Projects & tinkering

There's going to be changes that we'll make to suit our needs. By summer we aim to have the interior modified slightly for a trip away over the summer holidays.

Kids bunk bed

I'm building a bunk bed for the kids above the main fixed double bed. We'd like to keep all sleeping areas in the back so we can still use the dinning area as normal in the evenings. I want the bunk to be easily stored away when not in use so its going to fold away into the wall like a Murphy Bed.

I've purchased a fold-away bed which I'll be using for parts, and I've had to take away the cupboards above the main double bed to make space. I'll also be moving the T.V. to the other side of the bus so a bit of rewiring is going to need to happen.

GPS tracker installation

I've been pretty casual with parking my bus in places which aren't the safest, for a little bit more piece of mind I've purchased a GPS tracker for the bus.

It's a fairly simple device which will alert me if it moves, or by sending it a SMS it will let me know its location.

Purchased off AliExpress for next to nothing, so far its been easy to setup. I have it being tracked on a map, it has it's own battery incase its disconnected from the bus battery.
