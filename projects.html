Kids bunk bed

Plans are underway to create a bunk bed for the kids above the main fixed double bed.

I've purchased a fold-away bed which I'll be using as the parts, and I've taken away the cupboards above the main bed to make more space.

We'd like to keep all sleeping areas in the back so we can still use the dinning area as normal in the evenings. I want the bunk to be easily stored away when not in use so its going to fold away into the wall like a Murphy Bed.

For this all to happen I'll also be moving the T.V. to the other side of the bus so a bit of rewiring is going to need to happen.
